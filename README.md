### Step 3: Set Environment Variables

| Variable | Description |
|----------|-------------|
| `SESSION_ID` | Session ID from the pairing site (required) |
| `OWNER_NUMBER` | Your WhatsApp number (without `+`) |
| `BOT_NAME` | Bot name (default: `REDXBOT302`) |
| `PREFIX` | Command prefix (default: `.`) |

## ✨ Features

- 🔥 **Tag all group members** – `.tagall`
- 📱 **Auto‑status** – automatically view, react, and reply to statuses
- ⌨️ **Auto‑typing & recording** – simulates human interaction
- 🚫 **Anti‑delete, anti‑link, anti‑bad words** – keep groups safe
- 👑 **Group admin tools** – kick, add, promote, demote, mute, unmute, invite, revoke
- 📥 **Downloaders** – YouTube, Instagram, TikTok, Twitter, Pinterest, MediaFire, and more
- 🤖 **AI chat** – powered by multiple free AI models (Llama, Qwen, GPT)
- 🎨 **Sticker maker** – convert images/videos to stickers
- 🌦️ **Weather, quotes, facts, calculator, URL shortener** – daily utilities
- 📚 **100+ commands** – continuously updated
